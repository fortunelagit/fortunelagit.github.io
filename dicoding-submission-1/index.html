Memulai Project Pembuatan Website dengan Laravel Jetstream

Berikut langkah-langkah untuk memulai project dengan framework laravel Jetstream:

  1. Buka folder xampp/htdocs, kemudian jalankan console. (Jika tidak ada console yang mendukung, install dulu GitBash.)
  2. Jalankan perintah laravel new project --jet pada console.
  3. Kemudian pilih opsi antara 2 opsi, menggunakan framework Livewire atau Inertia. (Pada tutorial ini disarankan memilih framework Livewire.)
  4. Untuk opsi-opsi selanjutnya, pilih sesuai kebutuhan. Tunggu hingga hasil instalasi selesai.
  5. Setelah instalasi library selesai, project dapat mulai dikerjakan.

Mengubah Parameter Login dari Jetstream Default Login

Berikut langkah-langkah untuk mengubah parameter login dari Jetstream default login.

  1. Buka folder project yang sudah dibuat. (Project harus menggunakan Livewire, bukan Inertia.)
  2. Pergi ke folder app/Providers kemudian buka file JetstreamServiceProvider.php.
  3. Kemudian tambahlan kode berikut di fungsi boot. 
    Fortify::loginView(function () {
    return view('auth.login');
});

Fortify::authenticateUsing(function (Request $request) {
    $user = User::where('username', $request->username)
                ->first();

    if ($user && $request->password == $user->password) {
        return $user;
    }
});
  4. Ganti username dengan atribut lain yang diinginkan. (Misal: NIK, NPWP)
  5. Kemudian pergi ke folder selanjutnya, yaitu configdan buka file fortify.php.
  6. Hapus instruksi berikut: 'email' => 'email', dan ubah 'email' pada 'username' => 'email',dengan variabel yang diinginkan.
  7. Lakukan juga perubahan name dan type dari input pada file resources\views\auth\login.blade.php.
